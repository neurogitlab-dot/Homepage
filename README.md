# Portfolio Website

Eine moderne, statische Portfolio-Website fuer GitHub Pages.

## Seiten

- `index.html` - Startseite mit Kurzprofil
- `kompetenzen.html` - Kompetenzen, Technologien und Mehrwert
- `kontakt.html` - Kontaktbereich mit Platzhaltern fuer echte Links

## GitHub Pages

Dieses Projekt ist fuer dieses Repository vorbereitet:

`https://github.com/neurogitlab-dot/Hompage`

Nach dem Upload ist die Website voraussichtlich hier erreichbar:

`https://neurogitlab-dot.github.io/Hompage/`

Falls GitHub Pages nicht automatisch startet:

1. Auf GitHub das Repository `neurogitlab-dot/Hompage` oeffnen.
2. Unter `Settings` -> `Pages` als Quelle `GitHub Actions` auswaehlen.
3. Unter `Actions` pruefen, ob der Workflow `Deploy GitHub Pages` erfolgreich gelaufen ist.

## Anpassen

- In `kontakt.html` die Platzhalter fuer E-Mail, GitHub und LinkedIn/Xing ersetzen.
- Optional den Namen im Seitentitel und in den Texten ergaenzen.
- Neue Projekte koennen spaeter als eigene Sektion oder Seite hinzugefuegt werden.
